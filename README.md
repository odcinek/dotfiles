## dotfiles

### Dependencies (OS X)

```
brew install coreutils vim tmux wget
```

### Dependencies (Debian)
```
apt-get install vim tmux wget
```

### Embark
```
git clone git@github.com:odcinek/dotfiles.git ~/.dotfiles/
cd ~/.dotfiles && ./deploy.sh
```

### Cleanup
```
cd ~/.dotfiles && ./clean.sh
```
